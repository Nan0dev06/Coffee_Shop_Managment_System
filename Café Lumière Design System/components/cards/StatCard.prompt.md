# StatCard

Dashboard KPI card with a left accent gradient strip, metric label, large value, and optional trend indicator.

```jsx
<StatCard
  label="Total Revenue"
  value="$12,450"
  trend="up"
  trendLabel="+12% from last month"
/>

<StatCard
  label="Average Order Value"
  value="$8.50"
  showAccent={false}
/>
```
