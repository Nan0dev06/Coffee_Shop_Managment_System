# Café Lumière — Desktop App UI Kit

Interactive recreation of all 5 application screens in HTML/React.
Click sidebar items to navigate between screens.

## Screens
1. **Login** — Owner login with username/password
2. **Dashboard** — KPI stat cards, popular drink bar chart, recent orders
3. **Order Entry** — Customer selector, 5 drink cards, cart
4. **Inventory** — Ingredient stock table with low-stock badges
5. **Revenue Summary** — Date-scoped stat cards

## Notes
- This is a **web recreation** of the Java Swing UI for visual reference
- In production, use KControls (KButton, KGradientPanel) and XChart
- Icons are Unicode stand-ins — replace with PNG ImageIcon in Swing
