Interactive 5-screen desktop app recreation for Café Lumière — Login, Dashboard, Order Entry, Inventory, Revenue Summary.

Navigate between screens using the sidebar. Login with any username/password to enter the app.

Screens:
- **Login** — Full-screen dark-brown card with username/password inputs
- **Dashboard** — 4 stat cards, popular-drink bar chart, recent orders table
- **Orders** — Customer selector + 5 drink cards (Americano, Latte, Cappuccino, Espresso, Mocha)
- **Inventory** — Stock table with low-stock badges and restock buttons
- **Revenue** — Date picker, 4 stat cards, orders-for-day table
