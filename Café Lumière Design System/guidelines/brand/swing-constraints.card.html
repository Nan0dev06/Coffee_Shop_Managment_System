Use in Swing

  • Flat solid fills
  • 2-color linear gradients
  • Rounded rectangles
  • Solid 1–2 px borders
  • Instant hover/press color swaps
  • PNG images & icons

Avoid in Swing

  • Blurred shadows
  • Frosted glass / backdrop-filter
  • CSS transitions / animations
  • Multi-stop or radial gradients
  • Text shadows / glow
  • Semi-transparency / alpha
